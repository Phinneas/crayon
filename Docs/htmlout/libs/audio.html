Audio.Music

Encapsulates a sound file for the purpose of playing music

Audio.music.fadeIn

fadeIn(loop, inTransition)
Stops the currently playing music (if any) and fades in this music.
Args:
  • loop (bool): True if this music should loop indefinitely.
  • inTransition (float): Number of seconds to fade this music in

Audio.music.fadeOut

fadeOut(outTransition)
Fades out this music if it is currently playing. See also Music.fadeOutCurrent()
Args:
  • outTransition (float): Number of seconds to fade out

Audio.music.fadeOutAndIn

fadeOutAndIn(loop, outTransition, inTransition)
Fades out the currently playing music (if any) and fades in this music.
Args:
  • loop (bool): True if this music should loop indefinitely.
  • outTransition (float): Number of seconds to fade the current music out
  • inTransition (float): Number of seconds to fade this music in

Audio.music.fadeOutAndPlay

fadeOutAndPlay(loop, outTransition)
Fades out the currently playing music (if any) and starts playing this music.
Args:
  • loop (bool): True if this music should loop indefinitely.
  • outTransition (float): Number of seconds to fade the current music out

Audio.music.fadeOutCurrent

fadeOutCurrent(outTransition)
Fades out the current music, if any.
Args:
  • outTransition (float): Number of seconds to fade the current music out
