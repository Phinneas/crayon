directoryCreate(path, makeParents)
Creates a new directory

  • path (string): an absolute or relative path to create
  • makeParents (bool): true if parent directories should be created if already not present

FileIO.directoryExists

directoryExists(path)
Checks to see if a directory exists.
Args:
  • path (string): an absolute or relative path
Returns (bool): True if the path exists and is a directory (not a file).

FileIO.directoryList

directoryList(path, includeFullPath)
Returns a list of files and subdirectories in a given directory
Args:
  • path (string): an absolute or relative path
  • includeFullPath (bool): true if absolute file paths should be returned. Otherwise, file names only are returned.
Returns (list-of-string): list of files

FileIO.directoryMove

directoryMove(pathFrom, pathTo)
Moves a directory and its contents to a new location.
Args:
  • pathFrom (string): an absolute or relative path of a directory to move
  • pathTo (string): an absolute or relative path of a location to move a directory to

FileIO.fileCopy

fileCopy(pathFrom, pathTo, allowOverwrite)
Copies a file to a new location. Optionally allows overwriting files.
Args:
  • pathFrom (string): an absolute or relative path of the original file location
  • pathTo (string): an absolute or relative path of the new file location
  • allowOverwrite (bool): true if overwriting existing files is okay
